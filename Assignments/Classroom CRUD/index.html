Classroom CRUD

Function, objects, and elements, oh my!

What to build?

We are building a screen that will allow us to enter in basic student information

Once we enter in a student, it should get added to a list of students that is visible on the screen

All users should be visible on the screen and we should have Create, Read, Update and Delete capabilities

Must haves
  • An object that represents a student
    (first name, last name. email)
  • Create, Read, Update and Delete capabilities
  • UI screen must reflect any changes
  • Console log when applicable
Nice to haves

These are not required but if excellent progress is made and a challenge is desired

  • Validating data before student creation
  • Reset our form data after entering in a user
  • Separate presentation layer from update
  • Fancy icons
